#Installation Guide 
---
1. The requirement of python version is 3
2. Install packages in requirement.txt by using pip3
3. Run service.py in service folder
4. Run client.py in client folder
5. Go to web browser enter address: http://127.0.0.1:5001/
6. The system will set user name and password automatically.
7. The return data will show in the left scroll view in thw Webpage.
